# Rock Paper Scissors

A console-based Rock Paper Scissors game built with vanilla JavaScript as part of [The Odin Project](https://www.theodinproject.com/) Foundations Course.

## About

This project implements the classic Rock Paper Scissors game, played entirely in the browser console. The game pits a human player against the computer in a best-of-five match.

## Features

- Random computer choice generation
- Case-insensitive user input
- Score tracking for both players
- 5-round game format
- Winner announcement after each round and at game end

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Play

1. Open `index.html` in your web browser
2. Open the browser's developer console (F12 or right-click → Inspect → Console)
3. The game will prompt you to enter your choice for each round
4. Enter "rock", "paper", or "scissors" (case-insensitive)
5. Play 5 rounds and see who wins!

## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

